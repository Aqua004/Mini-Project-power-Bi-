🔷 About Project

Sales Data Analysis & Forecasting Dashboard is an interactive Power BI solution designed to analyze historical sales performance and predict future trends. The dashboard enables users to explore key business metrics such as sales, profit, and quantity across different dimensions like category, region, segment, and time.

The project focuses on transforming raw retail data into actionable insights using data visualization and forecasting techniques.

🔷 Objectives
Analyze sales performance across multiple dimensions
Identify high-performing and low-performing categories
Understand profit distribution and regional trends
Provide data-driven insights for decision-making
Forecast future sales using time-series analysis
🔷 Dataset
Source: Global Superstore Dataset
Time Period: Multiple years of transactional data
Key Fields:
Order Date
Sales
Profit
Quantity
Category / Sub-Category
Region / Segment
🔷 Tools & Technologies
Power BI – Data visualization & dashboard creation
Power Query – Data cleaning & transformation
DAX (Data Analysis Expressions) – Calculations & KPIs
GitHub – Version control & project hosting
Vercel – Deployment & portfolio hosting
🔷 Key Features
📊 Interactive dashboard with slicers (Region, Category, Segment, Year)
📈 Time-based sales trend analysis
📦 Category & sub-category performance insights
💰 Profit analysis with gain/loss visualization
👥 Customer-level insights (Top customers)
🔮 Sales forecasting using Power BI forecasting model
🔷 Key Insights
Technology category generates the highest sales
Certain sub-categories show consistent losses (e.g., tables)
Western region contributes the most profit
Sales show an overall upward trend over time
Forecast indicates stable short-term growth with fluctuations
🔷 Forecasting

A time-series forecasting model is applied to predict upcoming sales trends based on historical patterns. This helps in:

Planning inventory
Improving decision-making
Anticipating demand
🔷 Conclusion

This dashboard demonstrates how business intelligence tools can convert raw data into meaningful insights. It highlights trends, identifies problem areas, and provides a forward-looking view through forecasting.

🔷 How to Use
Use slicers to filter data by region, category, segment, or year
Navigate across different pages (Overview, Profit Analysis, Customer Insights, Forecast)
Analyze charts to understand performance and trends
